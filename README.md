# 👨‍💻 Víctor Gómez Molina

Desarrollador full-stack en formación, especializado en el diseño e implementación de aplicaciones web con arquitectura modular, autenticación segura y persistencia eficiente. Experiencia en proyectos académicos orientados a la defensa técnica, con énfasis en la calidad del código, la gestión de errores y la reproducibilidad funcional.

## 🧠 Áreas de especialización

- **Desarrollo backend** con Node.js y Express, aplicando principios RESTful y control de flujo robusto.
- **Autenticación mediante JWT** (access y refresh tokens), con gestión segura de sesiones.
- **Modelado de datos** y persistencia en MongoDB, utilizando Mongoose para validación y relaciones.
- **Desarrollo frontend** con Angular y React, empleando formularios reactivos y componentes reutilizables.
- **Integración de Bootstrap** para interfaces adaptativas y funcionales.
- **Configuración avanzada de entornos de emulación** (RetroArch, Flycast, PPSSPP), optimizando compatibilidad y rendimiento.

## 📄 Filosofía de desarrollo

Mi enfoque se basa en la **claridad estructural**, la defensa académica del código y la **fidelidad funcional**.
